Introduction

This page will provide information to frequently asked questions on the Event Management System (EMS) version 44.1 at Stern.


EMS is composed of two main user interfaces:

  • EMS Desktop Client
  • EMS Web App


Version 43 vs. Version 44.1

This section will provide answers to frequently asked questions about the changes that have been made in EMS version 44.1 since the previous version 43.

Why did we upgrade?

There have been significant updates to the new version of EMS, including: enhanced system security, increased performance, increased usability, and newly added features. There is also a new back-end component, Platform Services, that allows for a wider range of system customizations.

What's the difference between this version and the old version?

The biggest change you'll notice in EMS version 44.1 is the new interface of the EMS Web App, the replacement product for VirtualEMS. Please see the EMS Web App section for more information.

EMS Desktop Client

This section will provide answers to frequently asked questions about the EMS Desktop Client in version 44.1.

What is the EMS Desktop Client?

The EMS Desktop Client is software that is installed directly on your desktop workstation that allows access to all functions of EMS. Plesae note some functions of the EMS Desktop Client require specific permissions to access, that an EMS administrator can provision with applicable business case. Please see contacts on the bottom of this page for information on who you can send EMS access requests.

How do I download/install the EMS Desktop Client?

EMS version 44.1 has introduced a new feature called EMS Desktop Client Web Deploy that allows you to download the EMS Desktop Client over the Stern intranet. This new feature also ensures you download the most up-to-date version of the Desktop Client. Please see below for the URL to download the EMS Desktop Client:

http://ems-web16-p1.stern.nyu.edu/EmsDesktopClientWebDeploy

EMS Web App (formerly Virtual EMS)

This section will provide answers to frequently asked questions about the EMS Web App, formerly Virtual EMS.

What is the EMS Web App?

The EMS Web App is the upgraded version of VirtualEMS. It provides all of the same functionality as VirtualEMS with an updated user interface.

How do I log onto the EMS Web App?

The login page for the EMS Web App can be found here: https://ems-web16-p1.stern.nyu.edu/EmsWebApp/
