Introduction

This page will provide information to frequently asked questions for the Event Management System (EMS) version 44.1 at Stern.


Version 43 vs. Version 44.1

This section will provide answers to frequently asked questions about the changes that have been made in EMS version 44.1 since the previous version 43.


What version of EMS did we upgrade to?

Stern was previously using EMS version 43. The EMS software vendor, Accruent, has launched its newest version of EMS which is version 44.1.


Why did we upgrade?

There have been significant updates to the new version of EMS, including: enhanced system security, increased performance, increased usability, and newly added features.


What's the difference between this version and the old version?

The biggest change you'll notice in EMS version 44.1 is the new interface of the EMS Web App, the replacement product for VirtualEMS. Please see the EMS Web App section for more information.


EMS Desktop Client

This section will provide answers to frequently asked questions about the EMS Desktop Client in version 44.1.


What is the EMS Desktop Client?

Users can access EMS via two different interfaces:

  • EMS Desktop Client
  • EMS Web App


The EMS Desktop Client is software that is installed directly on your desktop workstation that allows access to all functions of EMS, depending on the security templates assigned to your account. Please see contacts on the bottom of this page for further EMS access request contact information.


How do I download/install the new EMS Desktop Client?

EMS version 44.1 has introduced a new feature called EMS Desktop Client Web Deploy that allows you to download the EMS Desktop Client over the Stern intranet. This new feature also ensures you download the most up-to-date version of the Desktop Client. Please see below for the URL to download the EMS Desktop Client:
http://ems-web16-p1.stern.nyu.edu/EmsDesktopClientWebDeploy


What if I already have the old version (v43) of the EMS Desktop Client installed?

The old version (v43) of the EMS Desktop Client will need to be uninstalled from your workstation before installing the new version. The Stern IT Enterprise Infrastructure & Security team has written step-by-step instructions on how to do this.


EMS Web App (formerly Virtual EMS)

This section will provide answers to frequently asked questions about the EMS Web App, formerly Virtual EMS.


What is the EMS Web App?

The EMS Web App is the upgraded version of VirtualEMS. It provides access to the main functions of EMS via the web, and provides all of the same functionality as VirtualEMS with an updated user interface.


How do I log onto the EMS Web App?

The login page for the EMS Web App can be found here: https://emswebapp.stern.nyu.edu/.


Are there instructions on how to use the EMS Web App?

Yes, The EMS software vendor, Accruent, has provided Stern with a Quick Reference Guide for using the EMS Web App. Please contact the Stern Helpdesk to request a copy of the Quick Reference Guide.


Miscellaneous

This section will provide answers to frequently asked miscellaneous questions about EMS version 44.1.


Who do I contact if I need assistance with EMS?

For any questions reagrding EMS, please contact the Stern Helpdesk using the "Contact Helpdesk" link on the side navigation bar in the EMS Web App, or see the contacts section below for further contact information.


How can I get the "Contact Helpdesk" link to open a new Gmail email?

Google has written instructions on how to set Gmail as your default email application, so that the "Contact Helpdesk" link will automatically start a new Gmail email addressed to the Stern Helpdesk.
